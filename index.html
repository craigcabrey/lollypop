Lollypop

Lollypop is a GNOME music player.

It provides:

  • mp3/4, ogg and flac.
  • Genre/Cover browsing
  • Genre/Artist/Cover browsing
  • Search
  • Queue
  • Party mode
  • Replay gain
  • Cover art downloader
  • Context artist view

Changes in 0.8.17:

  • Fix bugs introduced by 0.8.16
  • Fix artist context view
  • Fix previous track in shuffle
  • Smooth switch between views
  • Fix all artists view
  • Fix compilations track titles

Depends on:

  • gtk3
  • intltool (make)
  • itstool (make)
  • python (make)
  • python-cairo
  • python-dbus
  • python-gobject
  • python-mutagen
  • python-sqlite

Installation:

  • Sources
  • Archlinux: Available in Aur
  • OpenSUSE
  • Fedora
  • Debian/Ubuntu => Unity users need to disable overlay scrollbars (and report bug to Canonical): gsettings set com.canonical.desktop.interface scrollbar-mode normal

Ubuntu user need to install Gtk 3.12 and Adwaita icons 3.14 for better experience
