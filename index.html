Lollypop is a new GNOME music playing application.

It provides:

  • mp3/4, ogg and flac.
  • Genre/Cover browsing
  • Genre/Artist/Cover browsing
  • Search
  • Queue
  • Party mode
  • Replay gain
  • Cover art downloader
  • Context artist view

Changes in 0.8.10:

  • Rewrite queue widget
  • Bugfixes

Depends on:

  • gtk3
  • intltool (make)
  • itstool (make)
  • python (make)
  • python-cairo
  • python-dbus
  • python-gobject
  • python-mutagen
  • python-sqlite
